# 淘宝店铺合规客服系统（Taobao Compliance CS）

项目包含后端（NestJS）与前端管理后台（React + Vite），用于演示和部署淘宝店铺合规客服系统。

快速开始（开发）:

- 安装依赖（根目录或分别到子项目）：

```cmd
cd /d "c:\Users\34378\Desktop\自动回复机器人\taobao-compliance-cs"
npm install
cd admin
npm install
```

- 启动后端（开发）:

```cmd
cd taobao-compliance-cs\server
npm run start:dev
```

- 启动前端（开发）:

```cmd
cd taobao-compliance-cs\admin
npm run dev
```

构建生产包:

```cmd
cd taobao-compliance-cs
npm run build
cd admin
npm run build
```
