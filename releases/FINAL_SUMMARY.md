## 🚀 快速部署指令

### 生产环境部署 (3 步)
```bash
# 1. 解压
tar -xzf taobao-cs-v20251202-prod.tar.gz
cd taobao-cs-prod/server

# 2. 安装和初始化
npm install --production
npm run init:db

# 3. 启动
npm run start:prod
```

### 访问地址
- 🌐 前端: http://localhost:5173
- 📡 后端 API: http://localhost:3000/api/v1
- 📋 健康检查: http://localhost:3000/health
